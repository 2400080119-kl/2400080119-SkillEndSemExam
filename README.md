# End-Semester Exam Project - Student Portal

## Description
This is the End-Semester Exam project for the college course. It contains HTML and CSS implementation of a student portal website.

## Files Included
- `index.html` - Main HTML file with the portal structure
- `style.css` - Complete styling and layout CSS
- `README.md` - Project documentation

## Features
- Responsive student portal design
- Modern gradient header design
- Navigation menu with multiple sections
- Project cards display
- Mobile-friendly responsive layout
- Professional styling with hover effects

## Project Structure
```
├── index.html
├── style.css
└── README.md
```

## Technologies Used
- HTML5
- CSS3
- Flexbox for responsive design

## How to View
1. Clone the repository
2. Open `index.html` in a web browser
3. View the styled student portal interface
